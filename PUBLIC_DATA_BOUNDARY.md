# Public Data Boundary

This repository is public. It must never contain private home address records or sensitive proof materials.

## Allowed

Public-facing API documentation, routing metadata, published contact notices, source-identifying mark notices, and approved public archive references may live here.

## Not Allowed

Do not commit or publish:

- home address or apartment information;
- driver license records;
- address proof records;
- private Sunbiz notices that contain sensitive details;
- banking records;
- tax records;
- payroll records;
- service-account JSON;
- API keys or credentials;
- raw legal proof packets;
- private identity records;
- medical records.
